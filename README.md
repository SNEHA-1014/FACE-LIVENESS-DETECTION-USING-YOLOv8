# 🚀 Face Liveness Detection using YOLOv8

## 📌 Overview
Face Liveness Detection is an essential part of modern authentication systems, helping secure both personal and professional data. Traditional face recognition systems can be vulnerable to spoofing attacks using photos, videos, or masks.  

This project uses **YOLOv8**, a state-of-the-art object detection model, to perform **real-time face liveness detection** with high accuracy and speed.

---

## 🌟 Features

- 🎯 **YOLOv8-Powered Detection**  
  Fast and accurate liveness detection using advanced deep learning.

- 🐍 **100% Python Implementation**  
  Fully developed in Python for flexibility and easy customization.

- 🔐 **Enhanced Security**  
  Protects against spoofing attacks such as fake images or replay videos.

- ⚡ **Real-Time Performance**  
  Optimized for quick and responsive authentication systems.

- 🛠️ **Easy Setup & Usage**  
  Simple installation steps and beginner-friendly workflow.
